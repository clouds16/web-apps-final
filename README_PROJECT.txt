to run project, might need to install some libraries and dependencies

front end:
    npm install 
    npm install react-router-dom


backend:
    npm install mongoose
    npm install mongodb
    npm install validator
    npm install express
    npm install requests



----  Run ------

front end: 
npm start (or yarn start)



backend :  
cd src/backend
node index.js

backend is integrated by proxy
